# pkg-tui

Standalone Rust package for the `pkg-tui` binary.

This crate is intentionally self-contained so it can be moved to a separate repository without pulling files from the parent `osutil` tree.

## Install from source

From the repository root:

```bash
cargo install --path core/tabs/linux/applications-setup/pkg-tui
```

This installs `pkg-tui` into your Cargo bin directory (typically `~/.cargo/bin`).
